    Introduction

    DeepLIFT (Deep Learning Important FeaTures) is an efficient method for computing importance scores in a neural network. DeepLIFT was originally presented in “Learning Important Features Through Propagating Activation Differences” by Shrikumar, Greenside, Shcherbina & Kundaje.

    DeepLIFT is implemented in Python, and can be used to compute importance scores for models built with Keras. DeepLIFT uses TensorFlow to perform the backend computations.

    Getting Started

    Installation

    # clone the deeplift repository
git clone https://github.com/kundajelab/deeplift.git 
# install from clone; "editable" flag = automatic updates from changes to the code
pip install --editable deeplift/

    DeepLIFT depends on Theano (>= 0.8) and numpy (>= 1.9). It can autoconvert models trained with Keras 0.3 – support for models built with newer versions of Keras is experimental.

    Quickstart

    import deeplift
from deeplift.conversion import keras_conversion as kc

# 1) Convert a Keras sequential model
deeplift_model = kc.convert_sequential_model(
                    keras_model,
                    mxts_mode=deeplift.blobs.MxtsMode.DeepLIFT)

# specify the index of the layer to compute the importance scores of
# typically the input layer (index of 0)
find_scores_layer_idx = 0

# 2) Compile the function that compoutes important scores
# for the multipliers, instead use deeplift_model.get_target_multipliers_func
deeplift_contribs_func = deeplift_model.get_target_contribs_func(
                            find_scores_layer_idx=find_scores_layer_idx,
                            target_layer_idx=-1)

# 3) Compute scores
scores = np.array(deeplift_contribs_func(task_idx=0,
                                         input_data_list=[X],
                                         batch_size=10,
                                         progress_update=1000))

    1) Convert a Keras sequential model:
    MxtsMode defines the method for computing importance scores; supported methods include:

    • DeepLIFT
    • Gradient
    • DeconvNet
    • GuidedBackprop
    • GuidedBackpropDeepLIFT (a hybrid of GuidedBackprop and DeepLIFT where negative multipliers are ignored during backpropagation)

    2) Compile the function that compoutes important scores:
    In the case of MxtsMode.DeepLIFT, the importance scores are also called “contribution scores”

    • For sigmoid or softmax outputs, target_layer_idx should be -2 (the default, see “a note on final activation layers”)
    • For regression tasks with a linear output, target_layer_idx should be -1 (last layer)

    3) Compute scores:

    • input_data_list is a list containing the data for different input layers (e.g., for MNIST, there is one input layer with with dimensions 1 x 28 x 28)
    • task_idx represents the index of the node in the output layer that we wish to compute scores (e.g., if the output is a 10-way softmax, and task_idx is 0, we will compute scores for the first softmax class)
